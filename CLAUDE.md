# CLAUDE.md — AI캠프 데모 프로젝트

> 이 파일은 Claude Code가 **자동으로 읽고 따르는** 매뉴얼입니다.
> 여러분의 CLAUDE.md도 이런 모양으로 자라날 거예요.

## 세이브 규칙

- **"세이브해줘"** → `git add . && git commit`
  커밋 메시지는 방금 한 작업을 한국어로 한 줄 요약
- **"되돌려줘"** → `git checkout .` (가장 최근 저장으로 복구)
- **"세이브 목록 보여줘"** → `git log --oneline`

## GitHub 규칙

- **"깃헙에 올려줘"** → 필요하면 저장소 생성하고 `git push`
- **"가져와줘 [URL]"** → `git clone [URL]`
- README.md 없으면 자동으로 만들어서 이 프로젝트 설명 넣기

## 에러 대응

- 에러가 나면 에러 내용을 **한국어로 설명**
- `ModuleNotFoundError` → 자동으로 설치
- 3번 시도해도 안 되면 다른 방법 제안

## 프로젝트 규칙 — 일감 관리 대시보드

### 기본 동작
- 파일은 `index.html` **한 파일**만 (CSS, JS 내장)
- 외부 라이브러리 금지 — 순수 HTML/CSS/JS
- 데이터는 브라우저 `localStorage`에 저장 (새로고침해도 유지)

### 디자인
- 색: 남색 `#1e3a8a` + 흰색 + 포인트 주황 `#f97316`
- 폰트: 시스템 폰트 (Pretendard 있으면 Pretendard)
- 모바일 반응형

### 기능
- 텍스트 한 줄로 카드 추가 (예: "거래처 A 샘플 보내기, 금요일까지")
- 날짜·업체 자동 인식 → 태그로 표시
- 카드 드래그로 순서 변경
- 완료 체크 → 취소선 + 흐려짐
- 삭제 버튼 (작게)

## 데이터 형식 원칙

- AI가 다루는 데이터는 **텍스트 우선**: `.md > .pdf`, `.csv > .xlsx`
- 저장 파일명은 소문자 + 하이픈: `order-list.csv`
