# AI캠프 데모 프로젝트

> AI캠프 원데이 수강생을 위한 맛보기 프로젝트.
> 1교시 Part 3에서 **"깃헙에서 가져와줘"** 실습용.

## 🎯 포함된 것

1. **index.html** — 브라우저 기반 일감 관리 대시보드
   - 텍스트로 카드 추가 · 마감일 자동 인식 · 로컬 저장
2. **CLAUDE.md** — 샘플 규칙 매뉴얼 (여러분의 CLAUDE.md 작성 견본)

## ▶️ 실행 방법

### 수강생용 (Claude Code가 알아서)

Claude 채팅창에 한 마디만:

```
이거 실행해줘
```

→ 브라우저에 대시보드 자동 뜸.

### 수동으로 열 땐

`index.html`을 더블클릭하면 브라우저에서 바로 열립니다.
